  4. Program 3.3 Write a program to calculate area of a triangle using Heron,s formula:(Python) 
    # (Hint: Heron's formula is given as: area = sqrt(s*(s-a)*(s-b)*(s-c))*0.5)

a = float(input("Enter the value of first side of triangle : "))
b = float(input("Enter the value of second side of triangle : "))
c = float(input("Enter the value of third side of triangle : "))
print(a,b,c)
s = (a+b+c)/2
area = (s*(s-a)*(s-b)*(s-c))**0.5
print("Area = " +str(area))
